Execution Models
Subagents Default
Focused, independent tasks. Each runs in its own context, reports results back to the control agent. No peer communication.
Best for: extraction, analysis, synthesis, report generation
Agent Teams When planner recommends
Parallel exploration with inter-agent messaging. Shared task list, self-coordination. Teammates challenge and cross-reference each other.
Best for: competing hypotheses, adversarial review, triangulation
