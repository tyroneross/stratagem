# Creating Presentations (PPTX)

Use `create_pptx` with layout types: `"title"`, `"bullets"`, `"table"`, `"blank"`.

## Slide Rules
- Title IS the conclusion ("Revenue grew 23%" not "Revenue Analysis")
- Max 6 bullets × 8 words per slide
- Max 1 chart + 1 table per slide
- Always add speaker notes with source citations

## Table Content Format
Use pipe-delimited format:
```
Metric | Q1 | Q2 | Q3
Revenue | $1.2B | $1.4B | $1.5B
```
